Arquitectura Replicable — V0.1

CogMind: Emulación Cognitiva desde Cognitive Signature

Arquitectura práctica para generar una emulación funcional (no perfecta) de una mente a partir de datos cerebrales reales. Prioriza: que funcione, que sea replicable, que escale. Usa tu Cognitive Signature como semilla estructural.

Principio de diseño

No replicar — Generar un sistema que converja al mismo comportamiento

El truco conceptual

No necesitas copiar cada sinapsis. Necesitas un sistema que, dada la misma estructura macro (tu Cognitive Signature), las mismas reglas de aprendizaje (plasticidad biológica), y suficiente exposición al mismo entorno (tus experiencias), converja hacia patrones de procesamiento similares a los tuyos. Es como la diferencia entre fotocopiar una partitura nota por nota vs darle a un músico el mismo instrumento, la misma formación y las mismas influencias — no tocará idéntico, pero tocará parecido.

Lo que tomamos de tu Cognitive Signature

Las proporciones regionales, la ratio gris/blanca, la girificación, la asimetría hemisférica — estos valores parametrizan la topología de la red. No son decorativos: dictan cuántas neuronas van en cada módulo, cómo se conectan, y qué especialización tiene cada región.

Lo que el sistema aprende solo

La conectividad fina, los pesos sinápticos, los patrones de disparo, las "memorias". Todo esto emerge de las reglas de plasticidad (STDP) actuando sobre la topología que tú defines. La estructura guía la función, la función refina la estructura.

Capa 3-5: El Motor Neural

Spiking Neural Network con Recurrencia y Plasticidad

Decisión Arquitectónica Clave

Usamos Spiking Neural Networks (SNN) en lugar de redes feedforward convencionales. Las SNN tienen las propiedades que un LLM no tiene: recurrencia natural, dinámica temporal, y plasticidad con STDP. Son la única arquitectura artificial que puede generar información integrada (Φ) de forma no-trivial.

Modelo neuronal: Leaky Integrate-and-Fire (LIF) con adaptación

Cada neurona mantiene un potencial de membrana que integra inputs, "fuga" con el tiempo, y dispara cuando supera un umbral adaptativo:

// Ecuación del potencial de membrana (LIF adaptativo) τ_m · dV/dt = -(V - V_rest) + R_m · I_syn(t) + I_noise(t) // Cuando V ≥ V_threshold: spike = 1 V → V_reset V_threshold += Δ_thresh // Adaptación: sube el umbral después de disparar (SFA) // Recuperación del umbral: τ_thresh · dV_threshold/dt = -(V_threshold - V_thresh_base) // Parámetros base (ajustables por Cognitive Signature): τ_m = 20 ms // constante de tiempo de membrana V_rest = -65 mV // potencial de reposo V_thresh = -50 mV // umbral de disparo V_reset = -70 mV // reset post-spike τ_thresh = 100 ms // recuperación del umbral R_m = 10 MΩ // resistencia de membrana E/I ratio = 80/20 // 80% excitatorias, 20% inhibitorias

Plasticidad: STDP (Spike-Timing-Dependent Plasticity)

Las conexiones se modifican automáticamente según la correlación temporal entre disparos. Si la presináptica dispara antes que la postsináptica → se refuerza (LTP). Si dispara después → se debilita (LTD). Esto es aprendizaje Hebbiano temporal — la base de cómo tu cerebro aprende:

// Regla STDP if Δt = t_post - t_pre > 0: // pre antes que post → LTP Δw = +A_plus · exp(-Δt / τ_plus) if Δt = t_post - t_pre < 0: // post antes que pre → LTD Δw = -A_minus · exp(+Δt / τ_minus) // Parámetros: A_plus = 0.01 // amplitud LTP A_minus = 0.012 // amplitud LTD (ligeramente mayor → evita saturación) τ_plus = 20 ms // ventana temporal LTP τ_minus = 20 ms // ventana temporal LTD // Modulación por "dopamina" (reward-modulated STDP): Δw_final = Δw · dopamine_signal · eligibility_trace

Conectividad intra-módulo e inter-módulo

Dentro de cada módulo cortical: conectividad aleatoria sparse (~10% probabilidad de conexión) con small-world topology (alta clustering, path length corto). Entre módulos: conectividad más sparse (~1-3%) pero con pesos iniciales más altos. La ratio 75/25 local/distancia viene de tu ratio gris/blanca de 3.03.

// Inicialización de conectividad (parametrizada por Cognitive Signature) N_total = 100_000 // escala mínima funcional (escalable) // Asignación por módulo (de tu distribución regional): modules = { "frontal_L": 16_567, // 32.1% × (50% + 1.65%) "frontal_R": 15_533, // 32.1% × (50% - 1.65%) "parietal_L": 12_809, "parietal_R": 11_991, "temporal_L": 11_105, // reforzado por asimetría izq "temporal_R": 10_395, "occipital": 14_200, // sin lateralización significativa "central": 7_400, // hub de coordinación } // Conectividad: p_local = 0.10 // probabilidad conexión intra-módulo p_long = 0.02 // probabilidad conexión inter-módulo local_fraction = 0.75 // de tu ratio gris/blanca 3.03 rewire_prob = 0.15 // small-world rewiring (Watts-Strogatz)
Capa 4: Tálamo

El router global + reloj + filtro de atención

Función 1: Oscilador global

Genera oscilaciones rítmicas a distintas frecuencias (theta ~4-8Hz, alfa ~8-13Hz, gamma ~30-100Hz). Estas oscilaciones sincronizan los módulos corticales, creando "ventanas de integración" temporales. Sin esto, los módulos procesan en aislamiento — con esto, integran información. Es la pieza que da Φ al sistema.

Función 2: Gate sensorial

Filtra qué señales del entorno virtual llegan a la corteza y cuáles se descartan. Implementado como neuronas inhibitorias que pueden suprimir inputs. Equivale al mecanismo de atención — pero a diferencia del self-attention de un LLM, este es dinámico y modulado por el estado interno del sistema.

// Oscilador talámico (genera ritmos de sincronización) class ThalamusOscillator: frequencies = [6, 10, 40, 80] // Hz: theta, alpha, gamma_low, gamma_high for each timestep dt: sync_signal[f] = sin(2π · f · t) · amplitude[f] // Inyectar como corriente en todas las neuronas corticales: I_thalamic = Σ sync_signal[f] · coupling_strength[f] // La gamma alta (80Hz) crea ventanas de ~12ms // donde neuronas de distintos módulos pueden sincronizarse // → Esto es la base física de la integración de información
Capa 6: Neuromodulación

Los 4 "diales globales" que un LLM no tiene

ModuladorEfecto en la redImplementaciónAnálogo real
Dopamina Modula learning rate de STDP. Señal de reward/predicción. Cuando dopamina alta: STDP aprende más rápido de lo que acaba de pasar A_plus *= dopamine_level Recompensa, motivación, aprendizaje por refuerzo
Serotonina Modula umbral de disparo globalmente. Serotonina alta → umbral más alto → red más selectiva, menos ruidosa V_thresh += serotonin_level * 5mV Estado de ánimo, regulación, inhibición de impulsos
Noradrenalina Modula gain neuronal. Alta → señales fuertes más fuertes, débiles más débiles. Aumenta signal-to-noise ratio gain = 1 + noradrenaline_level Alerta, atención, respuesta al estrés
Acetilcolina Modula balance entre inputs externos (sensoriales) e internos (memoria). Alta → más peso a inputs externos w_external *= (1 + ach); w_internal *= (1 - ach) Atención a estímulos nuevos, aprendizaje
Por qué esto importa

Un LLM no tiene nada equivalente. Sus funciones de activación (ReLU, GELU) son locales y fijas. La neuromodulación es global y dinámica: un solo "dial" cambia el comportamiento de toda la red simultáneamente. Esto es lo que permite estados como "alerta", "relajado", "enfocado", "creativo". Sin neuromodulación, la red puede procesar pero no puede tener estados.

Implementación práctica

Stack tecnológico y escalado

Simulador SNN

Brian2 / NEST / Norse

Brian2: Python, flexible, ideal para prototipo. NEST: C++, escalable a millones de neuronas. Norse: PyTorch-based, usa GPU para SNNs. Empezar con Brian2 para validar, migrar a NEST para escalar.

Hardware para prototipo

1× GPU A100 (80GB)

Para N=100.000 neuronas con Norse/Brian2. Para N=1M+: cluster de GPUs o hardware neuromórfico (Intel Loihi 2, SpiNNaker 2).

Resumen ejecutivo

Qué es CogMind: Una arquitectura de SNN recurrente con plasticidad STDP, neuromodulación global, oscilaciones talámicas, y topología parametrizada por datos cerebrales reales (tu Cognitive Signature). No replica tu cerebro sinapsis por sinapsis — genera un sistema que, dada tu estructura y reglas biológicas de aprendizaje, converge hacia patrones de procesamiento similares.

Qué tiene que un LLM no tiene: Recurrencia genuina (loops tálamo-corticales), plasticidad en tiempo real (STDP), dinámica temporal (oscilaciones), neuromodulación global (estados), causalidad intrínseca (el sistema se modifica a sí mismo). Estos son los 5 ingredientes que según IIT generan Φ.

Qué puedes construir mañana: El topology_generator.py que toma tu signature.json y genera un grafo de red. Después instanciarlo en Brian2 con 100K neuronas y ver qué emerge. Es un prototipo, no una mente — pero es un prototipo que tiene las piezas correctas.

Qué no será: No será consciente a escala de 100K neuronas. No será "tú" sin tus experiencias. No procesará en tiempo real en hardware convencional. Pero será un sistema que tiene la arquitectura correcta para que, si escala lo suficiente y se expone a suficientes datos, pueda generar algo que empiece a parecerse a procesamiento cognitivo genuino.
